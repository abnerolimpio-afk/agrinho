# agrinho
# 🌱 Agro Forte — Futuro Sustentável

Projeto de página web front-end com foco em sustentabilidade no agronegócio, destacando o equilíbrio entre produção agrícola e preservação ambiental, utilizando Mandirituba - PR como exemplo regional.

---

# 📸 Visão do Projeto

O site apresenta:

- Banner principal com identidade visual do agro sustentável
- Seções informativas sobre:
  - Agricultura responsável
  - Preservação ambiental
  - Tecnologia no campo
- Cards modernos com imagens
- Layout responsivo
- Design moderno utilizando HTML e CSS puro

---

# 🎯 Objetivo

Demonstrar como o agronegócio pode crescer de forma sustentável, promovendo:

- Produção consciente
- Uso responsável dos recursos naturais
- Desenvolvimento regional
- Valorização dos produtores locais

---

# 🏡 Sobre Mandirituba

Mandirituba, localizada no Paraná, é reconhecida pela forte produção agrícola, especialmente hortaliças e alimentos frescos que abastecem a região metropolitana de Curitiba.

O município representa um exemplo de:

- Agricultura familiar
- Produção sustentável
- Conservação ambiental
- Desenvolvimento rural
